# 项目概述 python468

本项目是一个基于Python的协同过滤算法的电影推荐系统，采用Django框架进行后端开发，Vue.js进行前端开发。它包含了电影信息的展示、个人中心、搜索和收藏电影以及评分等丰富多样的功能。
